3. 数据流向图
═══════════════════════════════════════════════════════════════════════════════

用户操作 → 前端组件 → API客户端 → 后端服务 → 数据库 → 返回数据 → 更新UI
                         ↓                                    ↑
                   WebSocket ←─────→ 实时推送服务 ←────────────┘


典型请求流程：

1. 用户登录流程
   ┌────────┐   输入账号密码   ┌────────┐   POST /auth/login   ┌────────┐
   │ 用户   │ ──────────────→ │ 前端   │ ──────────────────→ │ 后端   │
   └────────┘                 └────────┘                      └────┬───┘
                                  ↑                                │
                                  │         返回 JWT Token          │
                                  │   ←─────────────────────────────┤
                                  │                                 │
                                  │                           验证密码
                                  │                           查询数据库
                              保存Token                            │
                              到 Store                            ↓
                                                          ┌────────────┐
                                                          │  MySQL DB  │
                                                          └────────────┘

2. 发帖流程
   ┌────────┐   提交帖子   ┌────────┐   POST /posts + JWT   ┌────────┐
   │ 用户   │ ───────────→│ 前端   │ ────────────────────→│ 后端   │
   └────────┘             └────────┘                       └────┬───┘
                              ↑                                 │
                              │    返回帖子数据                  │
                              │  ←──────────────────────────────┤
                              │                                 │
                          更新UI缓存                      验证Token
                          刷新帖子列表                    保存帖子
                              ↑                                 ↓
                              │                          ┌────────────┐
                              │                          │  MySQL DB  │
                              │                          └────────────┘
                              │                                 │
                              │       WebSocket推送新帖          │
                              │  ←──────────────────────────────┘
                              │
                      其他用户收到新帖通知

推荐部署方案：
  • 前端：Vercel / Netlify / 阿里云OSS静态托管
  • 后端：阿里云ECS / AWS EC2 / Docker容器化部署
  • 数据库：阿里云RDS / 自建MySQL主从
  • 缓存：阿里云Redis / 自建Redis集群
  • 文件：阿里云OSS / AWS S3
  • CDN：阿里云CDN / Cloudflare
