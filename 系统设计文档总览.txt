系统概述
═══════════════════════════════════════════════════════════════════════════════

校园论坛系统 (SchoolForum) 是一个基于 React + TypeScript 的现代化校园社区平台，
提供帖子发布、评论互动、私信聊天、用户关注、实时通知等核心功能。

主要特性：
  • 实时通信：WebSocket 支持实时消息推送和在线状态
  • 富文本编辑：支持 Markdown 和图片上传
  • 智能推荐：基于算法的内容推荐
  • 权限管理：完善的用户角色和权限控制
  • 响应式设计：适配多种设备屏幕

核心功能模块
═══════════════════════════════════════════════════════════════════════════════

【前台功能】
  ✓ 用户注册登录（邮箱验证、密码找回）
  ✓ 帖子浏览（分类筛选、标签筛选、热门推荐）
  ✓ 帖子发布（富文本编辑、草稿管理）
  ✓ 评论互动（评论、回复、点赞）
  ✓ 社交功能（关注、粉丝、私信）
  ✓ 搜索功能（帖子搜索、用户搜索）
  ✓ 实时通知（点赞、评论、关注、私信通知）
  ✓ 个人中心（资料管理、帖子管理、收藏管理）

【后台功能】
  ✓ 用户管理（用户信息、权限设置、封禁解禁）
  ✓ 内容管理（帖子审核、编辑、删除、置顶）
  ✓ 评论管理（审核、删除、屏蔽）
  ✓ 公告管理（系统公告发布）
  ✓ 服务中心（二手交易、学习资源）

技术架构概览
═══════════════════════════════════════════════════════════════════════════════

前端技术栈：
  • React 18 + TypeScript
  • React Router v6 (路由)
  • Zustand (状态管理)
  • React Query (数据请求)
  • Axios (HTTP客户端)
  • WebSocket (实时通信)
  • Tailwind CSS (样式)

后端技术栈：
  • Node.js / Nest.js (推测)
  • MySQL/PostgreSQL (数据库)
  • Redis (缓存)
  • Socket.io (WebSocket)
  • JWT (身份认证)

核心业务流程
═══════════════════════════════════════════════════════════════════════════════

1. 用户注册登录
   注册 → 邮箱验证 → 登录 → Token存储 → 权限验证

2. 发帖流程
   编辑内容 → 保存草稿/发布 → 内容审核 → 存储数据库 → WebSocket推送

3. 互动流程
   浏览帖子 → 点赞/收藏/评论 → 后端处理 → 更新统计 → 通知推送

4. 私信流程
   选择用户 → 建立对话 → WebSocket连接 → 实时聊天 → 已读回执

5. 通知流程
   事件触发 → 生成通知 → WebSocket推送 → 前端显示 → 用户点击 → 跳转页面

数据库设计概览
═══════════════════════════════════════════════════════════════════════════════

核心数据表：
  • users (用户表)
  • posts (帖子表)
  • comments (评论表)
  • messages (私信表)
  • notifications (通知表)
  • likes (点赞表)
  • favorites (收藏表)
  • follows (关注表)
  • drafts (草稿表)
  • announcements (公告表)

关键关系：
  • 用户 1:N 帖子
  • 用户 1:N 评论
  • 帖子 1:N 评论
  • 用户 N:N 用户 (关注关系)
  • 用户 1:N 通知

安全机制
═══════════════════════════════════════════════════════════════════════════════

前端安全：
  • XSS防护 (DOMPurify)
  • CSRF防护 (Token验证)
  • 输入验证

后端安全：
  • JWT认证
  • 权限控制 (RBAC)
  • SQL注入防护
  • 密码加密 (bcrypt)
  • 请求限流

通信安全：
  • HTTPS加密
  • WSS安全连接
  • Token刷新机制
